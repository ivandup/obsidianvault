# Google Dorking
Only shows web results from a domain, i Google search, enter
```
site:"microsoft.com"
```

Exclude "www." entries
```
site:"microsoft.com" -site:"www.microsoft.com"
```
Notice the "-site", which will exclude "www.microsoft.com"

## Search site for specific files with terms
```
site:"microsoft.com" filetype:ppt "penetration testing"
```

This will return any power point results from the Microsoft site with the term "penetration testing"

## Searching for signatures
Example, search for VNC viewer
```
intitle:"VNC viewer for Java"
```

Example: specific path (Mobotix Camera: /control/userimage.html)
```
inurl:"/control/userimage.html"
```

phpMyAdmin signature without password:
```
inurl:.php? intext:CHARACTER_SETS,COLLATIONS intitle:phpmyadmin
```

Search for compromised machines:
```
intitle:"-N3t" filetype:php undetectable
```
